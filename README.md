# webPawinhand_copysite

Spring Framework와 Oracle DB를 사용해 동물 입양 플랫폼 **포인핸드(Pawinhand)**의 핵심 기능을 모사한 카피 사이트 프로젝트입니다. 실서비스의 화면 구조와 사용자 흐름을 참고하여, 유기동물/입양 공고 조회·상세 보기·회원 인증 등 기본 플로우를 풀스택 기반으로 구현하는 것을 목표로 했습니다.

## 🎯 프로젝트 개요
- 포인핸드 UI/UX를 참고한 웹 애플리케이션 클론
- 유기동물/입양 공고 목록 및 상세 페이지 구현
- 회원 가입 / 로그인 등 기본 인증 흐름 연습
- Oracle DB 연동을 통한 실서비스와 유사한 데이터 처리 경험

## 🛠 기술 스택
- **Backend**: Java, Spring (MVC)
- **Frontend**: JSP, JavaScript, CSS
- **Database**: Oracle
- **Build & ETC**: Maven (pom.xml 기반)

## 💡 학습 포인트
- Spring MVC 기반 레이어드 아키텍처와 Oracle 연동 경험
- JSP + JavaScript 조합으로 리스트/상세/폼 화면 구성
- 실 서비스 구조를 분석해 DB 스키마와 화면을 재구성하는 연습
